<h2>About Me</h2>

I'm a passionate **Full-Stack Developer** with 4 years of professional experience in building modern web applications. I specialize in both frontend and backend development, working across the entire technology stack to create scalable, efficient, and user-friendly solutions.

### What I Do
- **Backend Development**: Java, Python (REST APIs, microservices, database design)
- **Frontend Development**: JavaScript, TypeScript, React (responsive UIs, state management)
- **Full-Stack Projects**: End-to-end application development from database to user interface
- **Database Design**: SQL optimization, data modeling, and cloud databases

### Tech Stack
- **Languages**: Java, JavaScript, TypeScript, Python
- **Frontend**: HTML5, CSS3, React, JavaScript/TypeScript
- **Backend**: Java, Python, Node.js
- **Databases**: SQL, PostgreSQL, MongoDB
- **Tools**: Git, REST APIs, Docker basics
